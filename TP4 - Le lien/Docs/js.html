1. Structure de base d’un programme

Fichier .js séparé

/* zone d'en-tête */
/* déclaration des variables globales */
let a;
let b;

/* définition des fonctions */
function maFonction() { }

/* CORPS DU PROGRAMME */
instruction1;
instruction2;

Il est conseillé de déclarer les variables en début de script.

JavaScript dans une page HTML

<script>
// variables globales et fonctions
</script>

La balise <script> permet d'ajouter du JavaScript.

4. Les variables

JavaScript est faiblement typé

let a = 3;
a = "coucou";

Types existants :

  • Number
  • BigInt
  • String
  • Boolean
  • Null
  • Undefined
  • Object
  • Symbol

Déclaration

let a;
const b = 4;
var c = "Bonjour";

On privilégie let.
