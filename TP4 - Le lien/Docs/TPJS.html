TP8 : Les Tableaux

Accueil | Retour

Introduction

Dans ce TP, il est demandé d’écrire des fonctions utilitaires permettant de travailler avec un tableau de taille fixe N. Ces fonctions seront réutilisables dans d'autres TP.

Le tableau utilisé contient N éléments. Il est interdit d’ajouter, d’insérer ou de supprimer une case.

let N = parseInt(prompt("Combien de case pour le tableau ?"));
let A = new Array(N);
let nbElemRempli = 0;

Méthodologie obligatoire

  1. Définir la boîte noire (entrées / sorties)
  2. Coder la fonction dans un fichier JS séparé
  3. Ajouter le mode d’emploi
  4. Tester avec l’inspecteur (console)

Exercice 1

Écrire une fonction initialiserTab1() permettant d'initialiser les N cases du tableau avec la valeur 0. À la fin, toutes les cases sont considérées comme remplies.

Exercice 2

Écrire une fonction initialiserTab2(valeur) permettant d'initialiser les N cases avec une valeur passée en paramètre. À la fin, toutes les cases sont remplies.

Exercice 3

Écrire une fonction remplirTab1(MIN, MAX) permettant de remplir les N cases du tableau avec des valeurs aléatoires comprises entre MIN et MAX. Les anciennes valeurs doivent être remplacées.

Exercice 4

Écrire une fonction remplirTab2(nb, MIN, MAX) permettant de remplir un certain nombre de cases à partir de la première case vide.

Si le nombre demandé dépasse la capacité restante du tableau, remplir uniquement jusqu’à la fin. Aucune case supplémentaire ne doit être ajoutée.

Exercice 5

Écrire une fonction afficherTab1() permettant d'afficher seulement les cases remplies du tableau.

Exercice 6

Écrire une fonction afficherTab2() permettant d'afficher toutes les cases du tableau, qu'elles soient remplies ou non.

Exercice 7

Écrire une fonction afficherTab3(DEB, FIN) permettant d'afficher les cases du tableau entre les positions DEB et FIN incluses.

Exercice 8

Écrire une fonction remplirElement(valeur) permettant de remplir la première case vide du tableau.

Si le tableau est complet, retourner false. Sinon retourner le nombre d’éléments remplis.

Exercice 9

Écrire une fonction saisirTab() permettant à l’utilisateur de saisir lui-même un certain nombre d’éléments.

Les valeurs sont ajoutées à partir de la première case vide. La fonction retourne le nombre de cases remplies.

Zone de test

Ouvrir la console (F12) pour tester les fonctions.
